# Sorting

## What is Sorting?

Sorting is the process of arranging elements in a specific order, usually:

- Ascending Order
- Descending Order

Sorting helps improve searching efficiency and data organization.

---

## Why Sorting?

Sorting is widely used in:

- Databases
- Search Engines
- E-Commerce Websites
- Banking Systems
- Data Analysis

---

## Real-Life Examples

- Arranging marks from highest to lowest.
- Sorting products by price.
- Sorting contacts alphabetically.
- Ranking students based on scores.

---

## Sorting Techniques Covered

1. Bubble Sort
2. Selection Sort
3. Insertion Sort
4. Merge Sort
5. Quick Sort

---

## Comparison

| Algorithm | Best | Average | Worst |
