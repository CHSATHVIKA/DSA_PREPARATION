This repository is organized around core problem-solving techniques that repeatedly appear in technical interviews and competitive programming.

⚙️ Core Techniques
1. Two Pointers

A technique used to optimize array and string problems by maintaining two indices that traverse the data structure in a coordinated manner.

It helps eliminate unnecessary iterations by reducing nested loops into a linear traversal.

Common use cases include searching pairs, in-place modifications, and working with sorted data structures.

2. Sliding Window

A pattern used to solve problems involving contiguous subarrays or substrings efficiently.

Instead of recomputing results for every range, a dynamic window is maintained and adjusted based on constraints.

It is widely used in problems involving maximum, minimum, or constrained subarray computations.

3. Prefix Sum

A preprocessing technique used to compute range-based queries efficiently.

By storing cumulative results, repeated calculations over subarrays can be reduced to constant time operations.

It is especially useful in sum-based query problems and array transformations.

Fast and Slow Pointers

A pointer-based technique used to detect cycles and solve linked list problems efficiently.

Two pointers move at different speeds to identify patterns such as repetition, intersection, or mid-point detection.

4. Recursion and Backtracking

A foundational technique used to explore all possible solutions through decision trees.

It is essential for problems involving permutations, combinations, subsets, and constraint-based exploration.

The approach builds solutions incrementally and backtracks when constraints are violated.
